# Formula-IT
В данном репозитории приведены результаты выполнения части лабораторных работ (Основы проектирования реляционных баз данных)  в рамках образовательной программы Formula-IT по направлению "Специалист по работе с базами данных".

В рамках курса были пройдены следующие темы: 
1. Основы проектирования реляционных баз данных
2. Основы языка SQL
3. Разработка БД PostgreSQL
4. Безопасная разработка
