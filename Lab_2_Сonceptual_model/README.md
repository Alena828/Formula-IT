Лабораторная работа 2. Создание концептуальной модели предметной области
Используя результаты анализа предметной области и упрощенную модель процессов:
1.	Составьте список сущностей предметной области
2.	Определите состав бизнес-атрибутов каждой сущности
3.	Для каждого атрибута определите:
    o	тип данных (обобщенно);
    o	признак обязательности (NOT NULL);
    o	пример допустимых значений (в соответствии с бизнес-правилами) атрибута;
    o	значение по умолчанию (если необходимо!)
4.	Для каждой сущности определите потенциальные ключи:
   o	выделите первичный ключ (в случае необходимости определите суррогатный первичный ключ);
   o	выделите альтернативные ключи
5.	Определите связи между сущностями и укажите характеристики связей (1:1, 1:М, M:N)
6.	Создайте ER-диаграмму предметной области
В качестве предметной области используйте БИБЛИОТЕКУ (см.лабораторная работа 1)
